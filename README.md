# EQ Banking Microservices Platform

A full stack banking platform inspired by digital-first banks.

This project showcases:

- Java Spring Boot microservices
- Angular dashboard
- REST APIs for accounts, transactions, and authentication
- Docker-based deployment
- CI/CD pipeline
- Monitoring & observability

## Structure

- backend/ → All microservices
- frontend/ → Angular client app
- gateway/ → API gateway
- docker/ → Docker-compose & containers
- docs/ → Diagrams & documentation
